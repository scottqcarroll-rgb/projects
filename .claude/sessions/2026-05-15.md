# Session Log — May 15, 2026

## What Was Built Today

### Daily Government Contract Report System (Brisar Investments LLC)

A fully automated daily email report that pulls from SAM.gov, filters by NAICS codes, scores contracts, and emails a formatted HTML report every morning at 8 AM.

**Script:** `govt-contracts/send_contract_report.py`
**Cron:** `0 8 * * *` — runs daily at 8 AM server time

---

## How It Works

1. Pulls up to **1,000 records** from SAM.gov API (one call per day)
2. **30-day window** (postedFrom = today minus 30 days)
3. Filters to **8 NAICS codes** across 4 categories
4. **Scores** each contract 1–10 based on actionability for Brisar
5. Saves raw JSON + categorized markdown to `govt-contracts/prospect-lists/`
6. Emails HTML report to `scottqcarroll@gmail.com`

## Scoring Logic (1–10)

| Factor | Points |
|---|---|
| Service-type title (maintenance, cleaning, repair, grounds, etc.) | +3 |
| Deadline 21+ days away (comfortable window) | +3 |
| Small Business Set-Aside | +2 |
| Contract value under $25K | +2 |
| Contract value $25K–$100K | +1 |
| Deadline 14–20 days away (tight but workable) | +1 |
| Has listed location | +1 |
| Deadline 7–13 days away (too tight to find sub) | −2 |
| Deadline under 7 days (not actionable) | −5 |

**Key reasoning:** Brisar needs at least 14 days to find a subcontractor, review scope, and submit a bid. Anything under 14 days is deprioritized.
