## Pages

### 1. Search (`/`) — 3-column layout
- **Left:** Filter panel — keyword, NAICS (all 13 Brisar codes), set-aside type, value ceiling, window, state, result limit
- **Center:** Results in Cards / List / Table view (switcher), score badges, urgent ⚠️ flags
- **Right:** Detail panel with 6 tabs:
  - **Details** — solicitation #, NAICS, value, set-aside, deadline, CO contact
  - **AI Analysis** — score breakdown, margin estimate (25-40%), USASpending pricing lookup
  - **Subcontractors** — placeholder for SAM entity radius search
  - **Requirements** — compliance checklist auto-generated per contract
  - **Templates** — 4 proposal templates (Cover, Consultant Agreement, Sub Agreement, Proposal)
  - **Proposal** → launches Proposal Wizard

### 2. Bid Tracker (`/bid-tracker`)
- 5-column Kanban: TRACKING → PREPARING → SUBMITTED → AWARDED → LOST
- Drag-and-drop cards, edit modal (notes + sub quote + move stage), delete
- State persisted to `data/bids.json`

### 3. Proposal Wizard (`/proposal-wizard`)
- 4-step stepper: Analyze → Write → Brand → Download
- Auto-fills proposal sections from form data and company profile
- Generates fully downloadable HTML proposal document
- Branding: 5 color options, contact info fields

## API Routes

| Route | Method | Description |
|-------|--------|-------------|
| `/api/search` | GET | SAM.gov contract search + enrichment + scoring |
| `/api/profile` | GET | Company profile |
| `/api/profile/sync` | POST | Pull from SAM.gov Entity API |
| `/api/bids` | GET/POST | Bid list / add new bid |
| `/api/bids/move` | POST | Move bid to new Kanban stage |
| `/api/bids/<id>` | PATCH/DELETE | Edit or delete bid |
| `/api/usaspending/pricing` | GET | Historical pricing from USASpending.gov |

---

## Key Behaviors

- **Search caching:** Results are cached by filter combination with date key — reruns same-day search without API call. Cache auto-expires after 3 days.
- **Auto-load:** Homepage auto-runs search using cached data on load.
- **Scoring:** Same 1–10 logic as daily report (service title, value, deadline, set-aside, location).
- **Margin estimate:** Detail panel shows projected 25–40% margin in AI Analysis tab.
- **Profile badge:** Shows Brisar entity status in header. Click ↻ to sync from SAM.gov.
