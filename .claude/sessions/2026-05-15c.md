## Key Architecture Decisions

### SAM.gov API Rules (never forget)
- **1,000 requests/day** for registered entities — resets at midnight UTC (hard cutoff)
- **Pagination burns quota** — each page = 1 call
- **No full text in API** — description is stub only; attachments require separate SAM.gov download
- **Strategy:** One broad daily pull → save 1,000 records → all filtering done locally

### USASpending.gov
- Completely free, no API key, no rate limits worth worrying about
- Max 100 records per page
- `award_type_codes` goes inside `filters`, not at root level
- `subawards: false` required to get prime contracts only
