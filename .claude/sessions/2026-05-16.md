## Architecture Notes

### SAM.gov API Rules
- 1,000 requests/day, resets midnight UTC
- `naicsCode` is NOT a valid entity search filter
- Agency names in opportunities use `fullParentPathName` (not `departmentName`)

### USASpending Rules
- Completely free, no API key
- Max 100 records per page
- Place of performance filter: `place_of_performance_locations: [{country: "USA", state: "XX"}]`
- Award URL format: `usaspending.gov/award/{generated_internal_id}/`
- Contract fields: use "Start Date"/"End Date" not "Period of Performance..."

### Odoo CRM
- URL: `https://100.124.71.12` | Admin / admin123
- DB auto-detected on first call
- Uses XML-RPC with SSL verification disabled (self-signed cert)
- Saves to `res.partner` (company contacts)
