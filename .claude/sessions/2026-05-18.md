## What Was Built

### Full Self-Hosted Website Stack
- **plastictravel.com** routed through Cloudflare Tunnel → clawz840 nginx → static files
- No static IP needed — Cloudflare Tunnel handles all routing outbound
- SSL managed automatically by Cloudflare (no cert maintenance)
- Both cloudflared and the Flask API run as systemd services (auto-start on reboot)

### 3-Page Static Site — `/var/www/plastictravel/`
- **index.html** — Hero (Santorini), stats strip, problem section (Venice), how it works (Tuscany), CTA
- **about.html** — About hero (Tuscany), two-col section (Amalfi), pricing model, CTA
- **contact.html** — Hero (Venice), contact form
- **style.css** — Shared dark navy/gold theme, responsive, frosted text backdrop on hero

### Contact Form → Odoo + Email
- Form fields: First Name, Last Name, Email, Notes
- POST `/api/contact` → Flask API on port 5051
- Creates a **Lead in Odoo CRM** (`odoo17_production`) via XML-RPC
- Sends **email notification to scottqcarroll@gmail.com** via Gmail SMTP (app password)
- Flask API: `/home/scott/projects/plastictravel-api/app.py`
- Credentials: `/home/scott/projects/plastictravel-api/.env` (gitignored)

### Infrastructure
- **cloudflared** — tunnel `plastictravel` (ID: `5a4d4298-97d2-4754-a106-984c20b6e414`)
- **nginx** — `/etc/nginx/sites-available/plastictravel` (port 80, proxies /api/ to 5051)
