## Key Insights from Transcript

### What a Claude Skill Is
- A markdown (.md) file — an SOP/recipe for Claude to follow
- One skill per specific task (granular beats broad)
- Transferable between Claude Code, Codex, Gemini, etc.
- Auto-updates based on feedback during use
- Stored in ~/.claude/skills/ (global) or .claude/skills/ (project)

### How to Create One
1. Open Claude on phone → voice dump your knowledge about a task
2. End with: "Use the skill creator skill"
3. Claude outputs a .md markdown file
4. Download it — that's your skill

### How to Sell
- Package multiple skills into a stack
- Build landing page in Claude Code (use a web designer skill)
- Add Stripe payment link
- Distribute via own website — no marketplace needed
- Soft CTAs in email newsletters, blog posts, social

### Ryan's Top 3 Skills (Most Popular)
1. SEO Blog Post Writer — turns YouTube video URL into ranked blog post in one prompt
2. YouTube Thumbnail Designer — uses GPT image model
3. Anti-Slop Skill — removes AI clichés ("delve", "robust", "I cannot", etc.)

### Pro Tips from Transcript
- Use markdown (.md) files — AI reads them better than PDFs
- "Triple check the skill instructions" trick reduces hallucinations
- Build skills you already use daily — then monetize them
- You can convert existing Claude Projects/GPTs into skills
- Karpathy auto-research = skill that auto-updates itself in background
